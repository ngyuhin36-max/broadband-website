唔同家庭情況嘅寬頻建議

1 個學生 + 家長上網

如果屋企得一個學生要上網課，加上家長平時上網睇片，100M（$98/月）已經足夠。就算小朋友上緊 Zoom 網課，家長同時睇 Netflix 都冇問題。

2 個學生同時上網課

兩個小朋友同時上 Zoom 課，加上家長 WFH 或者上網，建議 500M（$158/月）。100M 喺呢個情況下可能會有少少卡頓，特別係兩個都開緊視像嘅時候。

3 個以上學生 / 重度使用

如果屋企有 3 個以上學生，或者有學生要做影片 project、打機，1000M（$228/月）確保完全唔使擔心頻寬問題。

網課必備：確保唔會斷線嘅 5 個方法

  1. 用 LAN 線連接：如果小朋友用電腦上網課，用 LAN 線直接連 Router 比 WiFi 穩定好多
  2. 關閉其他裝置嘅下載：上網課時暫停其他裝置嘅更新同下載
  3. Router 擺近啲：將 Router 擺喺上網課嘅房間附近
  4. 用 5GHz WiFi：如果用 WiFi，連接 5GHz 頻段比 2.4GHz 快好多
  5. 準備手機熱點備份：萬一斷線，即刻用手機開熱點頂住

  • 共用情況：如果同 roommate 夾錢用，500M 會更適合多人使用
