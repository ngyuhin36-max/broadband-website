Mesh WiFi 全屋覆蓋教學：大單位救星

徹底解決 WiFi 死角問題，全屋每個角落都有靚 WiFi

2026 年 3 月 19 日 · 技術知識

你有冇試過喺廁所睇片突然 buffer、瞓房打機 lag 到爆、或者廚房個 smart speaker 成日斷線？如果你屋企有 WiFi 死角，Mesh WiFi 就係你嘅救星。呢篇文章由零開始教你認識 Mesh WiFi，幫你揀最適合嘅方案。

咩係 Mesh WiFi？

Mesh WiFi（網狀 WiFi）係由多個 WiFi 節點（Node）組成嘅無線網絡系統。唔同於傳統「一個 Router 撐全場」嘅做法，Mesh WiFi 將多個節點分散擺放喺屋企唔同位置，每個節點都識得同其他節點溝通，形成一個無縫嘅 WiFi 網絡。

你可以將 Mesh WiFi 想像成一隊接力跑手：第一個跑手（主節點）從 Router 接收信號，再傳俾下一個跑手（子節點），每個跑手都跑得同樣快，中間唔會掉棒。

幾時需要 Mesh WiFi？

如果你遇到以下情況，就應該認真考慮 Mesh WiFi：

  • 單位面積超過 700 呎：一個 Router 好難覆蓋全屋
  • 多房間 / 長走廊：牆壁會大幅削弱 WiFi 信號
  • 複式或村屋：唔同樓層需要獨立覆蓋
  • WiFi 死角明顯：某啲房間信號特別差
  • 成日要「追住 WiFi 走」：喺屋企行動時 WiFi 會斷線
  • Smart Home 裝置多：智能燈泡、IP Cam、掃地機械人等需要穩定 WiFi
  • WFH 需求：工作位離 Router 遠，WiFi 唔穩定

Mesh WiFi 點樣設定？

而家嘅 Mesh WiFi 設定已經好簡單，跟住以下步驟就搞掂：

Step 1：擺放主節點

將主節點用 LAN 線連接你嘅寬頻 Router（或者光纖 ONT）。主節點應該擺喺屋企嘅中心位置附近。

Step 2：下載 App

每個 Mesh WiFi 品牌都有自己嘅 App（例如 TP-Link Deco App、ASUS App 等），下載後跟住指示設定主節點。

Step 3：設定 WiFi 名稱同密碼

喺 App 入面設定你想用嘅 WiFi 名稱同密碼。建議用返同以前一樣嘅名稱同密碼，咁所有裝置唔使重新連接。

Step 4：擺放子節點

將子節點擺喺需要加強 WiFi 嘅位置。一般建議：

  • 兩個節點之間唔好隔超過 2 堵牆
  • 子節點同主節點之間要有一定嘅信號重疊
  • 放喺開放位置，唔好塞入櫃或者地面
  • 如果係複式/村屋，每層至少一個節點

Step 5：等待配對完成

子節點插電後，App 會自動偵測並配對，通常 2-3 分鐘就搞掂。配對完成後就可以即刻使用。

小貼士：設定完之後，可以用 App 嘅「網絡檢測」功能檢查每個節點嘅信號強度。如果某個節點顯示信號弱，試下移近啲主節點。BroadbandHK 客戶如果有設定問題，可以隨時 WhatsApp 我哋免費技術支援。

Mesh WiFi 節點擺位建議

700 呎以下兩房單位（2 個節點）

  • 主節點：客廳中間位置
  • 子節點：主人房或者走廊盡頭

700-1200 呎三房單位（3 個節點）

  • 主節點：客廳
  • 子節點 1：走廊 / 主人房
  • 子節點 2：最遠嗰間房

複式單位 / 村屋（3-4+ 個節點）

  • 每層至少 1 個節點
  • 節點盡量放喺樓梯附近，方便信號穿越樓層
  • 村屋可以考慮用有 Ethernet Backhaul 嘅 Mesh（用 LAN 線連接節點，速度更快）

雙頻 vs 三頻 Mesh：點揀好？

規格 雙頻 Mesh 三頻 Mesh
頻段 2.4GHz + 5GHz 2.4GHz + 5GHz + 5GHz/6GHz
節點間通訊 共用上網頻段 專用頻段，唔佔頻寬
速度損失 每跳約 30-50% 每跳約 5-15%
價錢 較平 較貴
建議 一般使用、預算有限 大單位、WFH、打機

如果預算許可，一定揀三頻 Mesh。因為三頻 Mesh 有專用頻段俾節點之間通訊，唔會搶佔你上網嘅頻寬，速度表現好好多。

Mesh WiFi 常見誤區

  • 「Mesh 越多節點越好」：唔係。節點太多反而會互相干擾，一般 700 呎用 2 個已經夠
  • 「有 Mesh 就唔使理 Router 位置」：主節點嘅位置仲係好重要，盡量擺中間
  • 「Mesh WiFi 可以突破寬頻速度」：Mesh 只係改善 WiFi 覆蓋，唔會令你嘅寬頻變快。如果寬頻得 100M，Mesh WiFi 速度唔會超過 100M
  • 「任何 Mesh 都一樣」：唔同品牌嘅 Mesh 效能差異好大，要揀信譽好嘅品牌
慳錢攻略：BroadbandHK 所有寬頻計劃免費送 WiFi Router。如果你嘅單位唔算太大（600 呎以下），我哋送嘅 Router 可能已經夠用。大單位先至需要另外買 Mesh WiFi。WhatsApp 查詢邊個方案最適合你。

常見問題

Mesh WiFi 同 WiFi extender 有咩分別？

WiFi extender（延伸器）只係將現有 WiFi 信號放大，會產生新嘅網絡名稱，切換時會斷線，而且速度通常會減半。Mesh WiFi 係多個節點組成一個統一網絡，用同一個 WiFi 名稱，裝置喺唔同節點之間會自動無縫切換，速度唔會明顯下降。簡單嚟講，Mesh 係「一個大 WiFi」，Extender 係「兩個細 WiFi 搭膊頭」。

Mesh Router 需要幾多個節點？

一般嚟講，700 呎以下嘅單位用 2 個節點已經夠；700-1200 呎建議 3 個節點；1200 呎以上或者複式單位可能需要 4 個或以上。村屋通常每層需要 1 個節點。具體要視乎間隔同牆壁材質，石屎牆比起輕巧間隔對 WiFi 影響大好多。如果唔確定，可以先買 2 件裝試用，之後再加。

Mesh WiFi 會唔會影響速度？

好嘅 Mesh WiFi 系統（特別係三頻 Mesh）對速度影響極小。三頻 Mesh 有專用頻段俾節點之間通訊，唔會佔用你上網嘅頻寬。雙頻 Mesh 可能會有少少速度損失（約 30-50%），但比起 WiFi extender 好好多。整體嚟講，Mesh WiFi 令到全屋任何角落都有穩定網速，比起只用一個 Router 然後遠距離收到好弱嘅信號好好多。
