慳錢提示：搬屋係轉換寬頻供應商嘅最佳時機！新客優惠通常比續約平20-40%，記得趁機格價比較。建議喺搬屋前3-4星期開始準備。

步驟一：查清楚舊寬頻合約狀況

搬屋前第一件事，就係搞清楚你而家嘅寬頻合約：

  • 合約到期日：打去供應商客服熱線或登入賬戶查詢
  • 提前解約費用：一般係剩餘月份 x 月費嘅某個百分比
  • 搬遷服務：大部分供應商都提供免費或低收費搬遷
  • 通知期：通常要30日前通知，唔好拖到最後一刻
重要：如果合約就嚟到期（3個月內），搬屋就係轉供應商嘅最佳時機。新客優惠通常比續約更抵！

步驟二：查詢新地址嘅寬頻覆蓋

唔同屋苑嘅寬頻覆蓋可以差好遠。有啲屋苑可能只有1-2間供應商，有啲就有5-6間任揀。你需要知道：

  • 新地址有邊幾間供應商覆蓋
  • 可以提供嘅最高速度（100M / 500M / 1000M / 2000M）
  • 係光纖到戶（FTTH）定光纖到大廈（FTTB）
  • 有冇特別嘅屋苑獨家優惠

最快捷嘅方法係透過BroadbandHK一次過查詢所有供應商覆蓋，唔使逐間打電話問。

步驟三：比較各供應商月費同優惠

知道邊幾間有覆蓋之後，就要逐間比較。比較嘅時候留意以下幾點：

  • 實際月費：留意係咪有頭幾個月優惠價，之後加價
  • 合約期：24個月定36個月？短約通常月費貴啲
  • 安裝費：大部分免費，但有啲供應商會收$380-$680
  • 附送設備：有冇送Wi-Fi路由器？質素點？
  • 搬屋優惠：部分供應商有專屬搬屋新客優惠

步驟四：簽約同預約安裝日期

決定咗供應商之後，盡早簽約同預約安裝：

  • 預約安裝一般需要3-7個工作天
  • 暑假、農曆新年後等搬屋旺季可能要等1-2星期
  • 安裝時間通常分上午（9:00-13:00）同下午（14:00-18:00）
  • 安裝當日需要有人喺屋企，工程約需30-60分鐘
貼士：盡量預約喺搬屋日或之後1-2日安裝。如果怕冇寬頻用，可以考慮先用手機熱點或申請臨時5G家居寬頻過渡。

步驟五：通知舊供應商搬遷或取消

處理舊寬頻有幾個選擇：

選擇A：搬遷服務（合約未到期）

如果合約仲有好長時間先到期，又唔想俾提前解約費，可以申請搬遷。供應商會將你嘅服務搬去新地址（前提係新地址有覆蓋）。搬遷費通常$0-$200。

選擇B：提前解約（想轉供應商）

如果想趁機轉去更平嘅供應商，可能要支付提前解約費。計一計數，如果新供應商嘅優惠夠大，可能解約都係抵嘅。例如：

  • 舊約剩餘6個月 x $178 = $1,068 解約費
  • 新供應商平$80/月 x 24個月 = 慳$1,920
  • 淨慳 $852，轉約係值得嘅！

選擇C：自然到期（合約就嚟完）

如果合約剛好到期，恭喜你！自由身可以完全無負擔咁格價轉會。

步驟六：安裝日準備工作

安裝師傅上門之前，做好以下準備：

  • 確認光纖入屋位置（通常喺客廳或工人房附近）
  • 清理光纖入屋點附近嘅雜物，方便施工
  • 準備好路由器擺放位置（盡量放喺屋企中央位置）
  • 如果係新裝修單位，提早同裝修師傅溝通預留線槽
  • 準備身份證同合約確認文件

步驟七：測試同優化新寬頻

安裝完成之後，記得做以下檢查：

  1. 速度測試：用有線連接做速度測試，確認達到合約承諾嘅速度
  2. Wi-Fi覆蓋：行勻全屋測試Wi-Fi信號，如果有死角考慮加Mesh路由器
  3. 設備連接：將所有智能家居設備重新連接新Wi-Fi
  4. 歸還舊設備：記得將舊供應商嘅路由器、機頂盒等設備歸還，避免被收費
  5. 確認舊約取消：打電話確認舊供應商已經正式取消你嘅服務

搬屋轉寬頻慳錢秘訣

把握以下幾個秘訣，搬屋轉寬頻可以慳唔少：

  • 善用新客優惠：新客月費通常比續約平20-40%，搬屋正好做新客
  • 格價唔好偷懶：同一間供應商喺唔同渠道嘅報價可以差$20-50/月
  • 留意隱藏費用：安裝費、路由器押金、提前解約費，通通計埋先比較
  • 問清楚合約條款：有啲優惠月費只係頭6個月，之後會加價
  • 手機寬頻組合：如果手機同寬頻揀同一間供應商，通常有額外折扣
  • 搬屋旺季避開：7-9月搬屋旺季，供應商優惠反而少啲。非旺季搬屋更易攞到筍價

常見問題 FAQ

搬屋寬頻可唔可以搬去新地址繼續用？

大部分供應商都提供搬遷服務，可以將現有合約搬去新地址繼續使用。但前提係新地址要有該供應商嘅覆蓋。搬遷通常免費或收取少量費用，建議直接聯絡供應商查詢。

搬屋寬頻合約未到期點算？

合約未到期有幾個選擇：1）搬遷服務將合約轉去新地址；2）提前解約並支付剩餘月費或罰款；3）轉讓合約俾新租客。建議先問清楚供應商嘅搬遷政策，搬遷通常係最抵嘅做法。

搬屋後幾耐先有寬頻用？

一般預約安裝需要3-7個工作天。如果係新樓入伙旺季（例如暑假），等候時間可能更長。建議喺搬屋前2-3星期預約安裝，確保搬入當日或之後幾日就有寬頻用。

搬屋轉寬頻邊間最抵？
