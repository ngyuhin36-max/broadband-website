❓ 國際金融中心 商業寬頻常見問題

IFC 係住宅定商廈？

IFC 係中環核心商業地段嘅甲級寫字樓綜合體，包括 One IFC（38 層，1998 年落成）同 Two IFC（88 層 / 415 米，2003 年落成）。非住宅，租戶主要係金管局、UBS、Nomura、Blackstone、Millennium 等金融機構及專業服務公司。

IFC 點搭車去？

IFC 商場直接連接港鐵香港站（東涌線 / 機場快線），並可經行人天橋步行至中環站（荃灣線 / 港島線）約 3 分鐘。亦有機場快線市區預辦登機服務。

IFC 有邊幾間商業寬頻供應商？

作為中環頂級甲廈，IFC 獲 HKBN Enterprise、PCCW / HKT Commercial、HGC Business、中信國際電訊 CPC、NTT、Telstra、Verizon 等主要 ISP 覆蓋，並支援企業光纖、MPLS、Cloud Connect、靜態 IP 及 SLA 保證。

IFC 裝商業寬頻要等幾耐？

IFC 由業主（置地 / 恒基 / 新地）統一管理，商業寬頻一般需 5-10 個工作天（視乎 ISP 同樓層接線），部分已接駁樓層可縮短至 3-5 天。

IFC 辦公室企業寬頻月費大約幾多？

基本商業 500M 約 $388-$588/月；1000M 企業光纖連 SLA + 靜態 IP 約 $888-$1,888/月；MPLS 專線 / Cloud Connect 另按方案報價。WhatsApp 5228 7541 提供樓層可獲實際報價。
