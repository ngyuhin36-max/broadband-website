屋企 WiFi 好慢？5 個即學即用嘅加速方法

明明裝咗高速寬頻，但 WiFi 仲係好慢？問題可能唔係出喺寬頻，而係 WiFi 設定。以下 5 個方法幫你即刻提升 WiFi 速度。

1. Router 擺放位置好重要

Router 應該放喺屋企正中間、高處、空曠嘅位置。避免放喺角落、地下或者被傢俬遮住。

2. 轉用 5GHz 頻段

大部分 Router 都有 2.4GHz 同 5GHz 兩個頻段。5GHz 速度更快，但覆蓋範圍較短。如果你離 Router 唔遠，記得連接 5GHz。

3. 定期重啟 Router

每個月重啟一次 Router，可以清除暫存記憶，改善連線速度同穩定性。

4. 更新 Router 韌體

廠商會定期推出韌體更新，修復問題同提升效能。記得定期檢查更新。

5. 升級 Router

如果你嘅 Router 用咗 3 年以上，考慮升級到支援 Wi-Fi 6 或 Wi-Fi 6E 嘅 Router。BroadbandHK 嘅 500M 同 1000M 計劃都免費送最新 Router。
